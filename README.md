# 2youdev Company Portfolio

A premium company portfolio website built with Next.js App Router, TypeScript, and Tailwind CSS.

## Tech stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- React

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Build for production

```bash
npm run build
npm run start
```

## Included pages

- Home
- About
- Services
- Projects
- Contact

## Notes

- Built with reusable components and a premium dark agency aesthetic.
- Contact details in the contact page are placeholders and should be replaced with real business information.
- Project examples and testimonials are sample content and can be customized.
